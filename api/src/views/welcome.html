Overview

Welcome to the Yamo API! This RESTful API provides financial management capabilities including:

  • Multi-book financial management
  • Account and transaction tracking
  • Categorization of income and expenses
  • Collaborative access for teams

Authentication

All API endpoints (except /health and this page) require authentication using JWT tokens.

To authenticate:

  1. Get a token by sending your credentials to POST /users/login
  2. Include the token in the Authorization header as Bearer <token>

Key Endpoints

POST/users/login

Authenticate and get an access token

GET/books

List all accessible books

GET/accounts

List accounts for a book

GET/transactions

Get transactions with filtering options

GET/categories

List categories for a book

GET/health

Check API service health
